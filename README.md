# Strudelfy

Creates a strudel.json file for a directory.

## Installing Linux

Download and unzip the binary and add it to your PATH variable for your shell.

You can use the `install` command in fish shell.

## Installing Mac

Download and unzip the binary and add it to your `/usr/bin` folder.

I do not have a mac to test on. If these instruction do not work please open an
issue.

## Installing Windows

Go to the [release page](https://github.com/CriusNyx/strudelfy/releases).

Download the newest release.

Unzip the file and put the strudel.exe file somehwere. I recomend making a
folder inside your users folder called bin.

ex `C:\users\yourUsername\bin`. Make sure to replace yourUsername with the name
of your users folder.

Add that folder to your PATH in environment variables.

## Using

Open up the command line in the folder you want to use.

use `strudelfy . https://github/User/Repo`. Make sure to replace the github url
with the URL for your gitrepo.
